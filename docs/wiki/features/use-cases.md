# Use Cases

moañete is not just a meeting tool. It listens to any audio your browser can capture and sees your screen.

## Meeting Assistant

Capture mic + tab audio. Get action items, key points, suggestions, and questions in real time.

- **Preset:** Meeting
- **Audio:** mic + tab (both sides of the call)
- **Screen:** optional (share slides for richer context)

## Lecture Note-Taker

Capture tab audio from a video call or recorded lecture. Get study notes, key concepts, and review questions.

- **Preset:** Lecture
- **Audio:** tab only (the lecture)
- **Screen:** share the slides/presentation for visual context

## Coding Interview Coach

During a live coding interview, get real-time coaching on your interview process, communication, and strategy.

- **Preset:** Code Interview (process coaching, no code hints)
- **Audio:** mic + tab
- **Screen:** share the coding environment

## LeetCode / Competitive Programming

Share your screen while solving problems. Get algorithm hints (Coach mode) or full solutions (Solve mode).

- **Preset:** LeetCode Coach (Socratic, never reveals answer) or LeetCode Solve (direct solution)
- **Audio:** mic (think aloud)
- **Screen:** share the problem + code editor

## Exam / Online Assessment Assistant

Share your screen during an online exam. Ask questions about what's displayed, get explanations.

- **Preset:** Q&A + screen capture
- **Audio:** mic (ask questions)
- **Screen:** share the exam/assessment page

## Pair Programming

Capture mic audio during a pairing session. Get bugs, design decisions, TODOs, and questions.

- **Preset:** Pair Programming
- **Audio:** mic (discussion)
- **Screen:** share the code

## YouTube / Podcasts / Audiobooks

Capture tab audio from any media. Get transcripts and summaries on demand.

- **Preset:** Q&A or Lecture
- **Audio:** tab only
- **Screen:** not needed

## Custom

Define your own insight categories and chat prompt for any scenario.

- **Settings:** custom `insightTabs` + custom chat prompt
- **Audio:** any combination
- **Screen:** optional
